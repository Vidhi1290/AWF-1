# Job Description — Backend Engineer (Node.js / Python)

**Company:** Anthrasync
**Location:** Bengaluru, India (Hybrid)
**Experience Required:** 3–6 years

## About the Role

We're looking for a Backend Engineer to help build and scale our automation
and data platform. You'll work closely with product and AI teams to design
APIs, data pipelines, and integrations that power our core product.

## Required Skills

- Strong proficiency in Node.js and/or Python
- Experience designing and building REST APIs
- Solid understanding of SQL and relational database design (Postgres or MySQL)
- Experience with Docker and containerized deployments
- Git-based version control workflows
- Familiarity with cloud platforms (AWS, GCP, or Azure)

## Preferred Skills

- Experience with workflow automation tools (n8n, Zapier, Airflow)
- Exposure to LLM/AI integration (OpenAI, Anthropic, or open-source models)
- Experience with message queues (RabbitMQ, Kafka, SQS)
- CI/CD pipeline experience (GitHub Actions, Jenkins)

## Education

Bachelor's degree in Computer Science, Engineering, or a related field
(or equivalent practical experience).

## Soft Skills

- Strong written and verbal communication
- Ability to work independently and take ownership of features end-to-end
- Comfortable in a fast-paced startup environment
